# Introduction

Source: extracted from `llms-full.txt` (`Introduction`).

## Overview

Effect is a TypeScript library for building synchronous and asynchronous programs with a consistent model.
It emphasizes composability, predictable error handling, and explicit control over side effects.

## Core Capabilities

| Capability        | What it gives you |
| ----------------- | ----------------- |
| **Concurrency**   | Fiber-based concurrency for scalable, low-latency applications. |
| **Composability** | Small, reusable building blocks for maintainable programs. |
| **Resource Safety** | Safe acquisition and release of resources, even on failure. |
| **Type Safety**   | Strong inference and explicit types for values, errors, and dependencies. |
| **Error Handling** | Structured failure management instead of ad-hoc exceptions. |
| **Asynchronicity** | A unified style for sync and async workflows. |
| **Observability** | Tracing support for debugging and runtime visibility. |
